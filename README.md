# SpaceX Launches
Provides a list of all SpaceX launches and specific information about each launch. 

## Built with
- Node.js
- React
- GraphQL
- Apollo
- Typescript
- Heroku
- [SpaceX API](https://docs.spacexdata.com/)

## Usage
```javascript
npm install
npm run start:dev
```

## Deployment
```javascript
npm run deploy
```
